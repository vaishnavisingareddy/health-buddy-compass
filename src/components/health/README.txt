Health components for One-Stop Health Friend:
- ConditionSelector: grid of condition cards
- ChatbotPanel: friendly, empathetic rule-based chatbot with quick chips, mic, TTS
- Tracker: water, sleep, steps, mood
- ResourceLibrary: simple condition-specific resources
- EmergencyInfo: urgent symptoms + map lookup
- CalmMode: breathing animation

All components use semantic tokens via utilities defined in src/index.css.
